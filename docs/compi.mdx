## Pre-requisite basic C++ :

1. Understanding of code flow
2. Knowledge of methods, variables, block, data types, function declaration.
3. Knowledge of import, return and define statements
4. Knowledge of basic if-else, for and while loops, switch case, pointers, static and dynamic memory allocation, ASCII codes, structures, iterators
